# Icons System

Tucu UI provides a comprehensive icon system that combines custom-designed icons with full Lucide React integration, giving you access to over 1,000+ high-quality icons for your applications.

## Icon Architecture

### **Dual Icon System**

Tucu UI implements a dual approach to icons:

1. **Custom Icons**: Carefully crafted SVG icons designed specifically for Tucu UI
2. **Lucide React Integration**: Complete access to the Lucide React icon library through the `LucideIcons` namespace

### **Lucide React Integration**

Tucu UI integrates the entire Lucide React library and exports it through a convenient namespace:

```typescript
// All Lucide icons are available through the LucideIcons namespace
import { LucideIcons } from 'tucu-ui';

// Access any Lucide icon using dot notation
<LucideIcons.Home />
<LucideIcons.User />
<LucideIcons.Settings />
```

This provides access to 5000+ icons through a single import without additional dependencies.

## Using Icons

### **Importing and Using Lucide Icons**

Import the LucideIcons namespace and access any icon:

```tsx
import { LucideIcons } from 'tucu-ui';

function MyComponent() {
  return (
    <div>
      <LucideIcons.Home className="w-6 h-6 text-gray-600" />
      <LucideIcons.User size={24} color="currentColor" />
      <LucideIcons.Settings className="w-5 h-5" />
      <LucideIcons.ChevronDown strokeWidth={1.5} />
      <LucideIcons.Search className="w-4 h-4 text-blue-500" />
    </div>
  );
}
```

### **Using Custom Tucu Icons**

Access custom-designed icons specific to Tucu UI through direct imports:

```tsx
import {
  Tucu,
  HomeIcon,
  SearchIcon,
  BookIcon,
  PlayIcon,
  InfoIcon,
  DocumentIcon,
} from 'tucu-ui';

function MyApp() {
  return (
    <div>
      <Tucu className="w-8 h-8 text-brand" />
      <HomeIcon className="w-6 h-6" />
      <SearchIcon className="w-5 h-5 text-gray-500" />
    </div>
  );
}
```

## Icon Props & Styling

### **Standard SVG Props**

All icons accept standard SVG attributes:

```tsx
import { LucideIcons } from 'tucu-ui';

<LucideIcons.Home
  className="w-6 h-6 text-blue-500 hover:text-blue-600"
  onClick={() => navigate('/')}
  aria-label="Go to home"
/>

<LucideIcons.User
  size={24}
  color="currentColor"
  strokeWidth={1.5}
  fill="none"
/>
```

### **Tailwind CSS Classes**

Icons work seamlessly with Tailwind utilities:

```tsx
// Size utilities
<LucideIcons.Home className="w-4 h-4" />     // 16px
<LucideIcons.Home className="w-6 h-6" />     // 24px
<LucideIcons.Home className="w-8 h-8" />     // 32px

// Color utilities
<LucideIcons.Home className="text-gray-600" />
<LucideIcons.Home className="text-brand" />
<LucideIcons.Home className="text-blue-500 hover:text-blue-600" />

// States
<LucideIcons.Home className="opacity-50" />
<LucideIcons.Home className="rotate-45" />
<LucideIcons.Home className="scale-110 transition-transform" />
```

### **Responsive Icons**

Create responsive icon sizes:

```tsx
<LucideIcons.Home className="w-4 h-4 sm:w-6 sm:h-6 lg:w-8 lg:h-8" />
```

## Best Practices

### **Icon Sizing**

Follow consistent sizing patterns:

```tsx
// Standard sizes
const iconSizes = {
  xs: 'w-3 h-3', // 12px - inline text
  sm: 'w-4 h-4', // 16px - small buttons
  md: 'w-6 h-6', // 24px - default size
  lg: 'w-8 h-8', // 32px - large buttons
  xl: 'w-12 h-12', // 48px - feature icons
};
```

### **Accessibility**

Always provide proper accessibility attributes:

```tsx
import { LucideIcons } from 'tucu-ui';

// Decorative icons
<LucideIcons.Home className="w-6 h-6" aria-hidden="true" />

// Interactive icons
<LucideIcons.Settings
  className="w-6 h-6 cursor-pointer"
  aria-label="Open settings"
  role="button"
  tabIndex={0}
/>

// Icons with text labels
<button>
  <LucideIcons.Home className="w-4 h-4 mr-2" aria-hidden="true" />
  Home
</button>
```

### **Performance Optimization**

Icons are automatically tree-shaken when using the namespace pattern:

```tsx
// ✅ Good - only imports the LucideIcons namespace
import { LucideIcons } from 'tucu-ui';

// Use any icon through the namespace
<LucideIcons.Home />
<LucideIcons.User />
<LucideIcons.Settings />

// ❌ Avoid - don't try to destructure individual icons
// This won't work with the current implementation
```

## Custom Icon Components

### **Creating Icon Wrappers**

Create reusable icon components:

```tsx
import { LucideIcons } from 'tucu-ui';

interface IconButtonProps {
  iconName: keyof typeof LucideIcons;
  label: string;
  onClick?: () => void;
}

function IconButton({ iconName, label, onClick }: IconButtonProps) {
  const IconComponent = LucideIcons[iconName];

  return (
    <button
      onClick={onClick}
      className="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
      aria-label={label}
    >
      <IconComponent className="w-5 h-5" />
    </button>
  );
}

// Usage
<IconButton iconName="Home" label="Home" onClick={() => navigate('/')} />
<IconButton iconName="User" label="Profile" onClick={() => navigate('/profile')} />
<IconButton iconName="Settings" label="Settings" onClick={() => setShowSettings(true)} />
```

### **Icon with Badge**

Combine icons with badges for notifications:

```tsx
import { LucideIcons } from 'tucu-ui';

function NotificationIcon({ count }: { count: number }) {
  return (
    <div className="relative">
      <LucideIcons.Bell className="w-6 h-6" />
      {count > 0 && (
        <span className="absolute -top-2 -right-2 bg-red-500 text-white text-sm rounded-full w-5 h-5 flex items-center justify-center">
          {count > 99 ? '99+' : count}
        </span>
      )}
    </div>
  );
}
```

## Integration Examples

### **Navigation Menu with Mixed Icons**

```tsx
import { LucideIcons, HomeIcon, PlayIcon } from 'tucu-ui';

const menuItems = [
  { icon: <HomeIcon />, label: 'Dashboard', href: '/dashboard' },
  { icon: <LucideIcons.User />, label: 'Profile', href: '/profile' },
  { icon: <LucideIcons.Settings />, label: 'Settings', href: '/settings' },
  { icon: <PlayIcon />, label: 'Media', href: '/media' },
  { icon: <LucideIcons.LogOut />, label: 'Logout', href: '/logout' },
];

function Navigation() {
  return (
    <nav className="space-y-2">
      {menuItems.map((item) => (
        <a
          key={item.href}
          href={item.href}
          className="flex items-center space-x-3 px-4 py-2 rounded-lg hover:bg-gray-100"
        >
          <div className="w-5 h-5">{item.icon}</div>
          <span>{item.label}</span>
        </a>
      ))}
    </nav>
  );
}
```

### **Status Indicators**

```tsx
import { LucideIcons } from 'tucu-ui';

const statusConfig = {
  success: { icon: 'CheckCircle', color: 'text-green-500' },
  error: { icon: 'XCircle', color: 'text-red-500' },
  warning: { icon: 'AlertCircle', color: 'text-orange-500' },
  pending: { icon: 'Clock', color: 'text-gray-500' },
} as const;

function StatusBadge({
  status,
  children,
}: {
  status: keyof typeof statusConfig;
  children: React.ReactNode;
}) {
  const { icon, color } = statusConfig[status];
  const IconComponent = LucideIcons[icon];

  return (
    <div className="flex items-center space-x-2">
      <IconComponent className={`w-4 h-4 ${color}`} />
      <span>{children}</span>
    </div>
  );
}
```
