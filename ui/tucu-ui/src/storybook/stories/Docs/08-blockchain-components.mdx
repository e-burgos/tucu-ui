# Blockchain Components

Tucu UI includes a specialized collection of blockchain and cryptocurrency components designed for DeFi applications, NFT marketplaces, crypto wallets, and Web3 interfaces. These components handle the unique requirements of blockchain applications while maintaining the same design consistency and accessibility standards.

## Component Documentation

### **CoinCard**

Display cryptocurrency information in a clean, card format with customizable background colors:

```tsx
import { CoinCard } from 'tucu-ui';

function CryptoPortfolio() {
  return (
    <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
      <CoinCard
        id="bitcoin"
        name="Bitcoin"
        symbol="BTC"
        logo="/icons/bitcoin.svg"
        balance="1.25"
        usdBalance="45,000"
        change="+2.5%"
        isChangePositive={true}
        color="#FDEDD4"
      />

      <CoinCard
        id="ethereum"
        name="Ethereum"
        symbol="ETH"
        logo="/icons/ethereum.svg"
        balance="10.5"
        usdBalance="33,600"
        change="-1.2%"
        isChangePositive={false}
        color="#E1F5FE"
      />
    </div>
  );
}
```

**Props:**

- `id`: Unique identifier for the coin
- `name`: Full name of the cryptocurrency
- `symbol`: Cryptocurrency symbol (e.g., BTC, ETH)
- `logo`: Path to the cryptocurrency logo image
- `balance`: Current balance amount
- `usdBalance`: USD equivalent value
- `change`: Price change percentage
- `isChangePositive`: Boolean indicating if change is positive or negative
- `color`: Optional background color for the card

### **CoinInfoCard**

Compact cryptocurrency information display with three size variants:

```tsx
import { CoinInfoCard } from 'tucu-ui';

function WalletBalance() {
  const coinData = {
    id: 'bitcoin',
    name: 'Bitcoin',
    logo: '/icons/bitcoin.svg',
    balance: '1.25 BTC',
    coinType: 'BTC',
  };

  return <CoinInfoCard item={coinData} variant="large" className="mb-4" />;
}
```

**Props:**

- `item`: Object containing coin information (id, name, logo, balance, coinType)
- `variant`: Size variant - 'small', 'medium', or 'large'
- `className`: Additional CSS classes
- `imageClassName`: CSS classes for the coin image

### **CoinListBox**

Searchable cryptocurrency selector with dropdown functionality:

```tsx
import { CoinListBox } from 'tucu-ui';
import { Bitcoin, Ethereum, Cardano } from 'tucu-ui/icons';

function SwapInterface() {
  const [selectedCoin, setSelectedCoin] = useState(coins[0]);

  const coins = [
    {
      icon: <Bitcoin />,
      code: 'BTC',
      name: 'Bitcoin',
      price: 45000,
    },
    {
      icon: <Ethereum />,
      code: 'ETH',
      name: 'Ethereum',
      price: 3200,
    },
    {
      icon: <Cardano />,
      code: 'ADA',
      name: 'Cardano',
      price: 0.65,
    },
  ];

  return (
    <CoinListBox
      coins={coins}
      selectedCoin={selectedCoin}
      setSelectedCoin={setSelectedCoin}
      disabled={false}
    />
  );
}
```

**Props:**

- `coins`: Array of coin objects with icon, code, name, and price
- `selectedCoin`: Currently selected coin object
- `setSelectedCoin`: Function to update the selected coin
- `className`: Additional CSS classes
- `disabled`: Boolean to disable the component

### **LivePriceFeed**

Real-time price display with integrated chart visualization:

```tsx
import { LivePriceFeed } from 'tucu-ui';
import { Bitcoin } from 'tucu-ui/icons';

function PriceTracker() {
  const priceData = [
    { name: 1, value: 45000 },
    { name: 2, value: 45200 },
    { name: 3, value: 44800 },
    { name: 4, value: 45100 },
  ];

  return (
    <LivePriceFeed
      id="bitcoin"
      name="Bitcoin"
      symbol="BTC"
      icon={<Bitcoin />}
      balance="1.25"
      usdBalance="45,000"
      change="+2.5%"
      isChangePositive={true}
      prices={priceData}
    />
  );
}
```

**Props:**

- `id`: Unique identifier
- `name`: Cryptocurrency name
- `symbol`: Cryptocurrency symbol
- `icon`: React element for the cryptocurrency icon
- `balance`: Current balance
- `usdBalance`: USD equivalent
- `change`: Price change percentage
- `isChangePositive`: Boolean for change direction
- `prices`: Array of price data points for the chart
- `isBorder`: Optional border styling

### **CurrencySwapIcons**

Visual indicators for currency swap operations with predefined cryptocurrency icons:

```tsx
import { CurrencySwapIcons } from 'tucu-ui';

function SwapDisplay() {
  return <CurrencySwapIcons from="ETH" to="USDC" />;
}
```

**Props:**

- `from`: Source currency code ('BTC', 'ETH', 'USDT', 'BNB', 'USDC', 'ADA', 'DOGE')
- `to`: Target currency code (same options as `from`)

**Supported Currencies:**

- BTC (Bitcoin)
- ETH (Ethereum)
- USDT (Tether)
- BNB (Binance Coin)
- USDC (USD Coin)
- ADA (Cardano)
- DOGE (Dogecoin)

### **NFTGrid**

Display individual NFT items with author information and collection details:

```tsx
import { NFTGrid } from 'tucu-ui';

function NFTGallery() {
  return (
    <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
      <NFTGrid
        author="John Doe"
        authorImage="/avatars/john-doe.jpg"
        image="/nfts/coolcat-1234.jpg"
        name="Cool Cat #1234"
        collection="Cool Cats"
        price="2.5 ETH"
        profilePath="/profile/john-doe"
      />
    </div>
  );
}
```

**Props:**

- `author`: NFT creator/owner name
- `authorImage`: Path to author's avatar image
- `image`: Path to the NFT image
- `name`: NFT name/title
- `collection`: Collection name
- `price`: NFT price
- `profilePath`: Optional path to author's profile

### **CollectionSelectList**

Searchable list component for NFT collection selection:

```tsx
import { CollectionSelectList } from 'tucu-ui';

function CollectionBrowser() {
  const collections = [
    {
      icon: '/collections/bayc-icon.jpg',
      name: 'Bored Ape Yacht Club',
      value: 'bayc',
    },
    {
      icon: '/collections/cryptopunks-icon.jpg',
      name: 'CryptoPunks',
      value: 'cryptopunks',
    },
  ];

  const handleSelection = (value: string) => {
    console.log('Selected collection:', value);
  };

  return (
    <CollectionSelectList
      collectionList={collections}
      onSelect={handleSelection}
    />
  );
}
```

**Props:**

- `collectionList`: Array of collection objects with icon, name, and value
- `onSelect`: Callback function when a collection is selected

### **TransactionInfo**

Simple component for displaying transaction details in a key-value format:

```tsx
import { TransactionInfo } from 'tucu-ui';

function TransactionDetails() {
  return (
    <div className="space-y-3">

```

**Props:**

- `label`: Label for the transaction detail
- `value`: Value to display (string or number)
- `className`: Additional CSS classes

## Design Considerations

### **Responsive Design**

All blockchain components are designed with mobile-first responsive principles:

- Cards stack vertically on mobile devices
- Sliders adapt breakpoints automatically
- Text sizes scale appropriately across screen sizes
- Touch-friendly interaction areas

### **Dark Mode Support**

Every component includes comprehensive dark mode styling:

- Automatic theme detection and switching
- Consistent color schemes across components
- Proper contrast ratios for accessibility

### **Performance Optimization**

- Lazy loading for images in NFT components
- Efficient chart rendering with Recharts
- Optimized slider performance with Swiper
- Minimal re-renders through proper prop handling

## Best Practices

### **Data Handling**

- Always validate cryptocurrency data before rendering
- Handle loading states appropriately
- Implement error boundaries for external data
- Use proper number formatting for currency values

### **User Experience**

- Provide clear visual feedback for state changes
- Use consistent iconography across components
- Implement proper loading indicators
- Handle edge cases gracefully

### **Accessibility**

- Include proper alt text for cryptocurrency icons
- Use semantic HTML structure
- Ensure keyboard navigation support
- Provide screen reader compatible content

The blockchain components in Tucu UI provide the essential building blocks for cryptocurrency and NFT applications, with a clear path for future expansion into comprehensive Web3 functionality.
